<details><summary>Details</summary>

 Most ML courses: theory → implementation. fast.ai: implementation → theory. You build a competitive image classifier in week 1. The "why" comes as you dig in. #### Part 1: Practical Deep Learning

 7 lessons. Covers: image classification, text classification, tabular data, collaborative filtering, language models, NLP. #### Part 2: Stable Diffusion from Scratch

 Advanced course. Builds Stable Diffusion from scratch in Python, explaining every component. Goes much deeper than the HuggingFace diffusion course. For when you want to really understand image generation.
 
#### Kaggle integration

 fast.ai students regularly top Kaggle competitions using the techniques from Part 1. After finishing the course: enter a Kaggle competition → it's directly applicable. A Kaggle gold medal from fast.ai techniques is an excellent credential.
 
#### Time commitment

 Part 1: 7 weeks at 8-10 hours/week. Run all notebooks on Kaggle (free GPU) or Colab. Don't rush — go back to previous lessons when concepts click. The forum (forums.fast.ai) is exceptional — active community who help debug.
 
#### Certification

 No formal cert — that's the point. fast.ai produces practitioners, not certificate holders. The proof is what you build.

## 🦜 LangChain / LangGraph Academy
**🔥 MUST USE** · _Agents, RAG, Production LLM Apps_

LangChain's official courses on building production LLM applications: RAG systems, agents, LangGraph for multi-agent workflows. Free and directly from the framework builders.

<details><summary>Details</summary>

#### LangChain Academy

 academy.langchain.com — free courses directly from LangChain. Covers: Introduction to LangGraph (the multi-agent framework), RAG from scratch, AI agents architecture, production deployment patterns. All Python, all production-focused, not toy examples.
 
#### LangGraph specifically (the most important)

 LangGraph is the serious multi-agent framework. It lets you define agent workflows as graphs (nodes = agents/tools, edges = routing logic). Supports: cycles (agents that loop), parallel branches, human-in-the-loop, state persistence across agent calls. This is what companies building real AI systems use.
 
#### RAG deep dive

 LangChain has a "RAG from scratch" series (YouTube playlist + notebooks). Goes from: naive RAG → advanced chunking → hybrid search → contextual compression → re-ranking → self-RAG (agents that decide when to retrieve). Each technique is a step up in quality.
 
#### What to build after

 Implement RAG with your own documents (notes, codebase, research papers). Add to Jarvis Research domain. Use LangGraph for the Jarvis orchestrator (routing between domains is a graph workflow). Build a document QA system as a portfolio project.
 
#### LangChain vs raw Anthropic SDK

 LangChain: more abstraction, faster prototyping, more integrations, larger community. Anthropic SDK: lighter, more control, better for Claude-specific features (extended thinking, caching). Learn LangChain for frameworks, use Anthropic SDK when you need precision.
 
#### Certification

 LangChain Academy issues completion certificates. Useful to list on LinkedIn but the code you write during the courses is more valuable as portfolio evidence.

</details>

Links: [LangChain Academy](https://academy.langchain.com)

#### AWS ML specialty

 After Cloud Practitioner: AWS Machine Learning Specialty. Validates: SageMaker, ML pipeline design, model deployment, data engineering for ML. More valuable than the generalist cert for ML roles. Study time: 2-3 months.

 Being able to: deploy a containerized app (ECS or Kubernetes), set up a CI/CD pipeline (GitHub Actions → ECR → ECS), configure proper IAM roles (principle of least privilege), set up CloudWatch monitoring. The cert validates these concepts but doing it is what gets you hired.

## 📊 Google Analytics + Data Studio
**✅ GOOD** · _Analytics, Dashboards, GA4_

Google Analytics 4 (GA4) + Looker Studio (formerly Data Studio) for tracking and visualizing web/app analytics. Free cert from Google. Useful for every app you launch.

<details><summary>Details</summary>

#### Why every developer should know GA4

 Every app you build needs analytics. GA4 is free, widely used, and hiring managers expect developers to know how to instrument and read analytics data. Knowing GA4 lets you: answer "what's actually working" vs "what feels like it's working", make data-driven product decisions, demonstrate end-to-end product thinking.
 
#### Google Analytics Certification

 Free at skillshop.withgoogle.com. Takes 4-6 hours. Covers: GA4 setup, event tracking, conversion tracking, audiences, reports, attribution modeling. Cert valid 1 year. The exam is 70 questions, 75% pass rate required.
 
#### GA4 implementation in Next.js

 npm install @next/third-parties → <GoogleAnalytics gaId="G-XXXXX" /> in layout.tsx. For custom events: gtag('event', 'button_click', {button_name: 'cta'}) . Track: page views, button clicks, form submissions, errors, purchase events.
 
#### Looker Studio (free dashboards)

 Connect GA4 → Looker Studio → build custom dashboards showing: traffic by source, conversion funnel, user retention, revenue by channel. Free, shareable links, auto-refreshes. Better than GA4's default reports for stakeholder presentations.
 
#### What to actually track

 For SaaS: signup rate, onboarding completion, feature adoption, churn indicators. For portfolio: time on page, scroll depth, contact form submissions, resume downloads. Track actions, not just page views — page views tell you nothing useful.
 
#### Privacy considerations

 GA4 uses cookies → requires cookie consent banner in EU (GDPR). For privacy-first alternative: Plausible Analytics ($9/month) — no cookies, GDPR compliant by default, simple dashboard.

</details>

## 👁️ Stanford CS231n + CS224n
**✅ GOOD** · _Computer Vision, NLP, Deep Theory_

Stanford's graduate ML courses, freely available online. CS231n (Computer Vision) and CS224n (NLP) are the gold standard for understanding deep learning theory at a rigorous level.

<details><summary>Details</summary>

#### CS231n — Convolutional Neural Networks for Visual Recognition

 cs231n.stanford.edu. Covers: CNNs from scratch, object detection, image segmentation, generative models (VAEs, GANs). The lecture notes are legendary — clearer than most textbooks. Assignments: implement backprop from scratch, build VGG-style CNN in NumPy, train ResNet on CIFAR-10. Takes 3-4 months if you do all assignments.
 
#### CS224n — Natural Language Processing with Deep Learning

 web.stanford.edu/class/cs224n. Covers: word embeddings (Word2Vec, GloVe), RNNs, Attention mechanism, Transformers (including the original "Attention Is All You Need" paper), BERT, GPT architectures. The transformer lectures explain exactly how ChatGPT and Claude work internally.
 
#### Who should take these

 If you want to: work at an AI research lab, do ML engineering at a top company, truly understand how modern AI works (not just use it). If you just want to build AI applications: fast.ai + LangChain courses are faster and more practical. CS231n/CS224n are for depth, not speed.
 
#### Math requirements

 You need: linear algebra (matrix operations, eigenvalues), multivariable calculus (partial derivatives, chain rule), probability (conditional probability, Bayesian inference). If rusty: 3Blue1Brown Linear Algebra series (YouTube) + Essence of Calculus series first.
 
#### Lectures availability

 All lectures on YouTube (search "CS231n 2024" or "CS224n Stanford"). Lecture notes and assignments on the course websites. No official cert — the knowledge is the value.

</details>

Links: [CS231n](http://cs231n.stanford.edu) · [CS224n](https://web.stanford.edu/class/cs224n)
